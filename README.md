## Installation
To get started create ``.env`` file with variables:

```
NoteClientId=
NoteClientSecret=
NoteURL=localhost:8080
```

API provides:
 
 - Google Login with oAuth 2.0
 - Redis as Primary Database

## Run

Run by: ``go run main.go``
